# Linux Basics Practice Scenarios 
---

## Scenario 1
You just logged into a Linux server. You want to check which directory you are currently in.  

**Command:**
```bash
pwd
```
---
## Scenario 2
You want to see all files and directories, including hidden files, in your current directory with details.

**Command:**
```bash
ls -al
```
---
## Scenario 3
You are deep inside some directory and want to go back to your home directory immediately.

**Command:**
```bash
cd
```
---
## Scenario 4
You want to move to the root directory of the filesystem.

**Command:**
```bash
cd /
```
---
## Scenario 5
You are in /etc/nginx/conf.d and want to go one level up to /etc/nginx.

**Command:**
```bash
cd ..
```
---
## Scenario 6
You moved to another directory and now want to go back to the previous directory you were in.

**Command:**
```bash
cd -
```
---
## Scenario 7
You want to see the username of the current logged-in user.

**Command:**
```bash
whoami
```
---
## Scenario 8
You want to see the system's hostname (the name of the computer/server).

**Command:**
```bash
hostname
```
---
## Scenario 9
You want to see the full system information, including kernel version, architecture, and OS type.

**Command:**
```bash
cat /etc/os-release
uname -a
```
---
## Scenario 10
You want to see the current date and time on the system.

**Command:**
```bash
date
```
---
## Scenario 11
You want to clear the terminal screen to have a clean workspace.

**Command:**
```bash
clear
```
---
## Scenario 12
You want to see all the commands you have executed so far in this terminal session.

**Command:**
```bash
history
```
---
